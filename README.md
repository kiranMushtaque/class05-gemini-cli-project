# Class 05 - GEMINI-CLI Assignment

This project is created as part of **Class 05** using **GEMINI-CLI**.  
The goal of this assignment was to replicate a design from a screenshot into a working webpage.  

## Features

- Fully responsive landing page
- Gradient background with curved header section
- Navigation bar with links and buttons
- Hero section with heading, subheading, description, and call-to-action buttons
- Designed using HTML & CSS (no frameworks)
